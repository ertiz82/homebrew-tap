# Homebrew Formulas

This repository contains Homebrew formulas for various tools.

## Available Formulas

| Formula | Description | Documentation |
|---------|-------------|---------------|
| [redgit.rb](redgit.rb) | AI-powered Git workflow assistant | [redgit.md](redgit.md) |

## Quick Install

```bash
# Add the tap
brew tap ertiz82/tap

# Install a formula
brew install <formula-name>
```

## Adding the Tap

```bash
brew tap ertiz82/tap
```

## Updating Formulas

```bash
brew update
brew upgrade <formula-name>
```
